Conectarea

Conectarea utilizatorilor

Pentru a accesa serviciile oferite de aplicatia Web Lego-HtmlCSS, trebuie sa va conectati utilizand Github. O data inregistrati in aplicatie, nu va mai fi nevoie sa va conectati ulterior, accesand direct serviciile oferite.

Dupa apasarea butonului de login veti fi redirectionati catre o pagina ce contine un form care va solicita informatiile necesare conectarii voastre prin Github. Completati acele casute si apoi dati login. Daca inregistrarea are loc cu succes, veti fi redirectionati catre pagina principal.

Utilizarea API REST-ului

Pentru a accesa intrebarile si raspunsurile nivelelor oferite de catre aplicatie web in cauza, utilizatorii se pot folosi de un link special creat pentru api rest-ul aplicatiei: https://lego-hmlcss.000webhostapp.com/app/views/API.php?cat=...&lvl=... unde cat poate fi hb,hi,he,cb,ci,ce (HTML,CSS-begginer, intermediate, advanced) iar lvl depinde de categoria initiala solicitata( hb-4,hi-4,he-3,cb-4,ci-6,ce-3).

Exemplu de apel: Html expert, nivelul 2

Meniul principal

Meniul principal este unul usor de utilizat. Acesta contine legaturi pentru a naviga catre pagina cu jocuri, statistici ori pagina profilului utilizatorului conectat. Pe ecranele mici, meniul este de tip hamburger, in dreapta. Recomandam citirea documentatiei pentru mai multe detalii.

Butoanele din meniul principal functioneaza astfel:

  • GAMES - trimite utilizatorul pe pagina de jocuri de unde acesta poate accesa nivelele atat pentru CSS, cat si pentru HTML.
  • STATISTICS - trimite utilizator pe pagina de statistici de unde acesta poate observa pozitia lui in intregul clasament al utilizatorilor.
  • My Statistics - trimite utilizatorul pe pagina cu statistici personale unde poate vedea progresul facut in ultimele sesiuni dar si procentul de exercitii finalizate din numarul total de exercitii.
  • PROFILE - trimite utilizatorul pe pagina profilului utilizatorului unde acesta poate sa observe detalii diverse despre punctele pe care le are dar si numarul de nivele efectuate cu succes.

Deconectarea

Pentru deconectare, un utilizator trebuie sa mearga pe pagina PROFILE si de acolo sa apese pe butonul LOGOUT.

Rezolvarea exercitiilor

Accesare nivelelor se face prin pagina GAMES. Pot fi accesate 3 tipuri de nivele: begginer, intermediate, advanced pentru 2 domenii: CSS si HTML.

Structura unui exercitiu pastreaza un template clar definit. In partea din stanga se afla numarul nivelului impreuna cu tipul acestuia si o scurta descriere a probelemei ce trebuie efectuate. Sub descrierea nivelului, se afla un form ce necesita completat pentru a efectua cu succes un nivel. In tot acest timp, in dreapta se afla o interfata grafica ce arata live modificarile efectuate peste formul din stanga.

Dupa completarea casutelor, se apasa pe butonul de CHECK VALIDITY pentru a verifica daca varianta propusa este corecta. In cazul in care aceasta este corecta, si asa ar fi bine sa fie, utilizatorul poate fi redirectionat catre urmatorul nivel. Altmiteri, trebuie sa incerce alta varianta.

Calcularea scorului

Scorul este actualizat de fiecare data cand se rezolva corect inca un exercitiu. Actualizarea se face in functie de corectitudinea si viteza raspunsului oferit.

Pentru a vedea modificarile, puteti accesa pagina MY STATISTICS ori PROFILE.
